# Team-Project

This repository contains the programs and outputs for the url shortner.

## Experiments Included

* Experiment 2: URL Shortener using Scrum Framework
* Experiment 3: Pair Programming (URL Shortener with Click Counter)
* Experiment 4: Calculator using Test Driven Development (TDD)

## Contents

* HTML program files
* Output screenshots for each experiment

## Description

Each program demonstrates the implementation of Agile concepts such as Scrum, Pair Programming, and TDD using simple web-based applications.
